API Key Management

HMAC-SHA256 authenticated API keys for programmatic trading.

How to Use

Headers:
X-API-Key: Your API key
X-Nonce: Current millisecond timestamp
X-Signature: HMAC-SHA256(METHOD + PATH + NONCE, secret_key)

Example (cURL):
NONCE=$(date +%s%3N)
SIG=$(echo -n "GET/api/v1/balances/alice$NONCE" | openssl dgst -sha256 -hmac "YOUR_SECRET" | cut -d' ' -f2)
curl -H "X-API-Key: YOUR_KEY" -H "X-Nonce: $NONCE" -H "X-Signature: $SIG" http://localhost:8080/api/v1/balances/alice
