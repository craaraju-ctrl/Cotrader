## Root Causes Found & Fixed
1. **LLM never used** — `LlmExecutor.is_ollama_running()` returned cached `false`. Fixed: probes Ollama live on each call.
2. **LLM responses empty** — nemotron-3-nano is a thinking model, output in `thinking` field not `response`. Fixed: `chat()` reads `thinking` as fallback.
3. **LLM prompts ineffective** — Information dumps didn't elicit BUY/SELL. Fixed: structured "You are a trading signal generator" prompts.
4. **HOLD treated as failure** — `available = action != "HOLD"`. Fixed: HOLD is now a valid LLM response.
5. **Embeddings dimension mismatch** — fastembed 384-dim vs server 768-dim. Fixed: switched to Ollama's `/api/embed`.
6. **Vector embeddings never stored** — Root cause was #5. Fixed with #5.
